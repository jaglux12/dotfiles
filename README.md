# Dotfiles by jaglux12

Configuration files. Simple and minimalist design, it does not have large aesthetic configurations. i currently use it on my main machine, it has an **Amd Ryzen 3 2200g** and 16gb ram, no gpu. Operating system is [GNU/Linux Mint 21](https://linuxmint.com/edition.php?id=299).

### Files

* [Awesome Window Manager](https://awesomewm.org/)
* [Conky](https://github.com/brndnmtthws/conky)
* [Fish](https://fishshell.com/)
* [Kitty](https://github.com/kovidgoyal/kitty)
* [Oh my fish](https://github.com/oh-my-fish/oh-my-fish)
* [Picom](https://github.com/yshui/picom)
* [Polybar](https://github.com/polybar/polybar)
* [Rofi](https://github.com/davatorium/rofi)
